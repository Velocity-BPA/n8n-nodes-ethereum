# Automate Ethereum & EVM Blockchains with n8n: Introducing n8n-nodes-ethereum

We're excited to announce the release of n8n-nodes-ethereum, a powerful community node that brings Ethereum and EVM blockchain automation directly into your n8n workflows!

## The Challenge

Blockchain developers and Web3 teams often struggle with integrating blockchain data into their existing business processes. Whether you're monitoring wallet transactions, tracking NFT transfers, interacting with smart contracts, or resolving ENS domains, these tasks typically require custom scripts, complex API integrations, or expensive third-party services.

## The Solution

Velocity BPA has created n8n-nodes-ethereum to bridge this gap. This comprehensive node brings native Ethereum and EVM-compatible blockchain functionality into n8n's visual automation platform, enabling you to build powerful Web3 workflows without writing code.

## Key Features

- **Account Management**: Query balances, track addresses, and monitor account activity across Ethereum and EVM-compatible chains
- **Transaction Handling**: Send, receive, and monitor blockchain transactions with full transparency
- **Smart Contract Interaction**: Read from and write to smart contracts using a simple interface
- **Token Operations**: Work with ERC-20 tokens, check balances, and handle transfers
- **NFT Integration**: Query NFT metadata, track ownership, and monitor collection activity
- **ENS Resolution**: Resolve Ethereum Name Service domains and reverse lookups seamlessly

## Real-World Use Cases

Imagine automatically triggering Slack notifications when a specific wallet receives a payment, syncing NFT sales data to your CRM, monitoring smart contract events and updating your database, or creating automated treasury management workflows. With n8n-nodes-ethereum, these scenarios become simple drag-and-drop operations.

## Getting Started

Installation is straightforward. In your n8n instance, simply run:


npm install n8n-nodes-ethereum


Restart n8n, and you'll find the Ethereum node available in your node palette, ready to integrate into your workflows.
